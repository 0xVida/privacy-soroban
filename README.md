## Ghxst: Privacy-Preserving Payments on Stellar

Ghxst is a protocol for absolute financial privacy on the Stellar network. it leverages zero-knowledge cryptography to enable shielded transactions that are verifiable, compliant and instantly settled

### Repository Structure

- [frontend/](./frontend): UI
- [backend/](./backend): orchestrator for proof generation and relay services
- [contracts/](./contracts): Soroban smart contracts for on-chain state and verification
- [circuits/](./circuits):ZK circuits for tx shielding
- [packages/shared/](./packages/shared): Shared types and cryptographic utilities
- [infra/](./infra): infra configurations
- [docs/](./docs): Technical specifications and protocol architecture
